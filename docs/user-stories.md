# User Stories

> Complete user stories for the FORGE project, organised by persona and feature area. All MVP stories (M-01 through M-58) are built. Phase 2 and Phase 3 stories are planned.

---

## Member: Onboarding

| # | Story | Phase |
|---|-------|-------|
| **M-01** | As a new user, I want to see an engaging splash screen with the FORGE wordmark and a progress bar so that I know the app is loading and I feel the brand's premium quality from the first moment. | MVP |
| **M-02** | As a new user, I want to swipe through 3 welcome slides that explain what FORGE is and why appearance is measurable so that I understand the value proposition before committing to anything. | MVP |
| **M-03** | As a new user, I want to answer a 7-question quiz about my goals, current routine, and preferences so that FORGE can personalise my 90-day transformation plan. | MVP |
| **M-04** | As a new user, I want to see my estimated starting FORGE Score after completing the quiz so that I get a psychological hook and understand my starting point before creating an account. | MVP |
| **M-05** | As a new user, I want to create an account with just my name, email, and password so that I can save my quiz answers and access my personalised plan. | MVP |
| **M-06** | As a returning user who already has an account, I want to log in with my email and password so that I can pick up where I left off without redoing the quiz. | MVP |
| **M-07** | As a new user, I want to see an animated loading screen with rotating coaching messages while my personalised 90-day plan is being generated by AI so that I understand the app is working and feel anticipation. | MVP |
| **M-08** | As a new user, I want to review my personalised 90-day plan with focus areas, expected improvements, and an AI-generated compliment before committing so that I feel confident this program is tailored to me. | MVP |
| **M-09** | As a new user, I want to see a clear paywall with the $11/mo price, feature list, and free trial so that I can make an informed purchase decision without pressure. | MVP |
| **M-10** | As a new user, I want the app to save my quiz progress as a draft so that if I close the app mid-quiz, I can resume where I left off instead of starting over. | MVP |

## Member: Daily Tasks

| # | Story | Phase |
|---|-------|-------|
| **M-11** | As a member, I want to see my daily tasks on the home screen when I open the app so that I know exactly what I need to do today to improve my appearance. | MVP |
| **M-12** | As a member, I want to tap a task and see its detail screen with the title, pillar, duration, XP value, and "why it works" explanation so that I understand the purpose behind each task. | MVP |
| **M-13** | As a member, I want to mark a task as complete by tapping a single button so that I get instant feedback and my progress is recorded. | MVP |
| **M-14** | As a member, I want to see my XP increase and my streak update immediately after completing a task so that I feel a sense of accomplishment and momentum. | MVP |
| **M-15** | As a member, I want my incomplete tasks to appear first, ordered from shortest to longest duration, so that I can tackle quick wins first and build momentum. | MVP |
| **M-16** | As a member, I want to see an "All Done" celebration with animated checkmark and XP summary when I complete all of today's tasks so that I feel rewarded and motivated to come back tomorrow. | MVP |
| **M-17** | As a member completing my first-ever task, I want to see a special First Strike celebration with a micro-score and AI compliment so that my first accomplishment feels significant and memorable. | MVP |
| **M-18** | As a member who missed several days, I want to resume at my last program day (not restart from day 1) so that my progress is preserved and I don't feel punished for life getting in the way. | MVP |

## Member: Progress & Scoring

| # | Story | Phase |
|---|-------|-------|
| **M-19** | As a member, I want to see my FORGE Optimisation Score prominently on the progress tab with a large, animated number so that I have a single clear metric for how my appearance is improving. | MVP |
| **M-20** | As a member, I want to see a breakdown of all 9 pillar scores in colored progress bars so that I understand which areas are improving and which need attention. | MVP |
| **M-21** | As a member, I want to see a 91-day heatmap showing which days I completed all tasks, partially completed, or missed so that I can visualise my consistency over time. | MVP |
| **M-22** | As a member, I want to see the delta (change) in each pillar score since my last cycle check-in so that I can connect my daily actions to measurable results. | MVP |
| **M-23** | As a member, I want to see my XP level (Initiate through Grandmaster) and how close I am to the next level so that I have a sense of long-term progression beyond daily tasks. | MVP |
| **M-24** | As a member, I want to understand what affects each pillar score so that I know which tasks matter most for the areas I care about. | MVP |
| **M-25** | As a member, I want to see my current and longest streak counts on the progress screen so that I'm motivated to maintain consistency and can celebrate my dedication. | MVP |

## Member: Cycle Check-Ins

| # | Story | Phase |
|---|-------|-------|
| **M-26** | As a member, I want to take a baseline photo on day 3 with a face guide overlay and clear instructions so that the app has a reference point to measure my progress against. | MVP |
| **M-27** | As a member, I want to take a progress photo every 3 days using a guided camera viewfinder with a face outline so that my photos are consistent in angle and framing. | MVP |
| **M-28** | As a member, I want the app to analyse my cycle photo with AI and return pillar scores, an overall FORGE score, and a primary insight so that I get objective, data-driven feedback on my appearance. | MVP |
| **M-29** | As a member, I want to view my cycle check-in history in a scrollable photo timeline so that I can see my transformation unfold visually over time. | MVP |
| **M-30** | As a member, I want to compare two cycle photos side-by-side with a draggable slider so that I can spot subtle differences between earlier and later check-ins. | MVP |
| **M-31** | As a member, I want to see pillar score trends graphed across all my cycle check-ins so that I can track which pillars are steadily improving and which have plateaued. | MVP |
| **M-32** | As a member, I want to receive a push notification every 3 days reminding me to take my cycle photo so that I don't forget and fall behind on my progress tracking. | MVP |

## Member: Coaching & Insights

| # | Story | Phase |
|---|-------|-------|
| **M-33** | As a member, I want to see a short daily AI coaching insight (≤25 words) on my home screen so that every day I get a fresh, personalised piece of guidance relevant to where I am in the program. | MVP |
| **M-34** | As a member, I want to receive a weekly coaching report every Sunday with pillar movement analysis, completion stats, and next week's focus so that I can reflect on my progress and plan ahead. | MVP |
| **M-35** | As a member completing Season 1, I want to receive a narrative season report in story format comparing my before/after scores and highlighting my biggest breakthroughs so that I can see the full arc of my transformation. | MVP |
| **M-36** | As a member, I want the coaching language to evolve as I progress — from outcome-focused early on, to habit language mid-program, to mechanism-level explanations later — so that the coaching stays relevant to my growing understanding. | MVP |
| **M-37** | As a new member, I want to receive an AI-generated compliment about my strongest pillar after my baseline photo is analysed so that I start the program with a confidence boost. | MVP |
| **M-38** | As a member, I want to be notified when my weekly report is ready so that I never miss the Sunday summary that anchors my weekly reflection. | MVP |

## Member: Gamification

| # | Story | Phase |
|---|-------|-------|
| **M-39** | As a member, I want to earn XP for every task I complete (10 base + 20 streak bonus when streak > 3) so that my daily effort is recognised and quantified. | MVP |
| **M-40** | As a member, I want to level up through 6 tiers (Initiate → Grandmaster) as my total XP increases so that I have a long-term progression system that goes beyond daily tasks. | MVP |
| **M-41** | As a member, I want to build a streak by completing tasks on consecutive days so that consistency is rewarded and I'm motivated to not miss a day. | MVP |
| **M-42** | As a member, I want to see milestone celebrations at 7, 30, 60, and 90-day streaks so that reaching streak thresholds feels like a real achievement. | MVP |
| **M-43** | As a member, I want to unlock badges by completing challenges (e.g., "7-day posture streak", "Complete all tasks for a week") so that I have variety beyond the daily routine. | MVP |
| **M-44** | As a member, I want to participate in 7-day micro-sprints focused on boosting a single pillar so that I can intensively target an area that needs improvement. | MVP |

## Phase 3: Intelligent Coaching & New Modalities

| # | Story | Phase |
|---|-------|-------|
| **M-66** | As a member with a stalled pillar, I want the AI to detect the stagnation and suggest a targeted 7-day micro-sprint so that I can proactively address plateaus before they become frustrating. | Phase 3 |
| **M-67** | As a member, I want to see a predicted score projection ("projected score in 14 days") based on my current trajectory and task completion patterns so that I can visualise where I'm heading. | Phase 3 |
| **M-68** | As a member, I want an AI voice-guided morning routine that walks me through my tasks in sequence with audio cues so that I can follow my routine hands-free while in the bathroom. | Phase 3 |
| **M-69** | As a member who uses a health wearable (Apple Watch, Whoop, Oura), I want FORGE to incorporate my real sleep and activity data into my recovery and nutrition pillar scores so that my biometrics contribute to my appearance scores. | Phase 3 |
| **M-70** | As a member, I want personalised habit-stacking suggestions that reorder my daily tasks based on when I'm most consistent so that my routine is optimised to my natural patterns. | Phase 3 |
